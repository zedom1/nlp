### VSM：

使用语料库 Reuters R8

使用模型 ： KNN、SVM

实现经典特征权重计算方法：

1. tf-idf
2. tf-chi
3. tf-rf
4. tf-ig
5. tf-eccd
6. iqf_qf_icf

实现论文提出的两种新的权重计算方法：

1. tf-dc
2. tf-bdc

整个实现过程详见[这篇文章](http://zedom1.top/2018/06/01/VSM/)
